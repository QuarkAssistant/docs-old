# User-facing tools

tscircuit is split across several user-facing tools. Use this page to choose the right one for the task you are working on.

## Online editor

Use the [tscircuit online editor](https://tscircuit.com/editor) to write, preview, and share circuit code in the browser. It is the quickest way to try tscircuit without installing anything locally.

## Package registry

Use the [tscircuit package registry](https://tscircuit.com/trending) to discover reusable circuit packages and examples from the community.

## Command-line interface

Use the [`tsci` command-line interface](https://github.com/tscircuit/cli) when you want to run tscircuit from a local project, automate builds, or integrate circuit generation into a development workflow.

## Preview and rendering tools

The tscircuit ecosystem includes focused tools for viewing generated circuit artifacts:

- [`pcb-viewer`](https://github.com/tscircuit/pcb-viewer) renders PCB views.
- [`3d-viewer`](https://github.com/tscircuit/3d-viewer) renders 3D board previews.
- [`runframe`](https://github.com/tscircuit/runframe) runs interactive examples and previews.

## Autorouting

Use [autorouting.com](https://autorouting.com/) and the related [`autorouting`](https://github.com/tscircuit/autorouting) packages when you need automated routing support for PCB layouts.

## Circuit JSON and converters

Circuit JSON is the interchange format used by tscircuit tools. These repositories help convert between tscircuit and other formats:

- [`circuit-json`](https://github.com/tscircuit/circuit-json) defines the shared data model.
- [`kicad-component-converter`](https://github.com/tscircuit/kicad-component-converter) helps convert KiCad component data.
- [`dsn-converter`](https://github.com/tscircuit/dsn-converter) handles DSN conversion workflows.

## Core packages

Use [`core`](https://github.com/tscircuit/core) and the other packages in the [tscircuit GitHub organization](https://github.com/tscircuit) when contributing to the underlying compiler, component model, and rendering pipeline.
